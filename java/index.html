# Java

  • == 与 equals：基本数据类型==比较的是值，引用数据类型==比较的是内存地址。Object里equals比较的是内存地址，其他类里重写的一般是比较内容。
  • final:
    • 变量：初始化后不能更改
    • 方法：锁定方法，private隐式指定为final
    • 类：不能被继承
  • 重载与重写
    • 重载：同一个类中，方法名相同，参数类型、个数、顺序有所不同，发生在编译时。
    • 重写：父(非private)子类中，方法名、参数列表必须相同，返回值、抛出异常范围小于等于父类，访问修饰符范围大于等于父类。
