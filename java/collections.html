# 集合框架

  • Collections

    • List：ArrayList、LinkedList。
    • Set：HashSet，LinkedHashSet，TreeSet，都是根据相应的Map实现的。
    • Map：HashMap，LinkedHashMap（HashMap基础上增加双向链表，保持插入或访问顺序），HashTable，TreeMap(红黑树)。

  • HashMap

    • 拉链法，链表数组，支持null键值，初始容量16，table数组使用数目超过负载量即进行2倍扩容。JDK1.8后加入红黑树、扩容机制等优化。

    • HashTable：方法都经过synchronized，不支持null。

  • ConcurrentHashMap

    • JDK1.7 分段数组+链表，采用可重入锁。

    • JDK1.8 数组+链表/红黑树(同HashMap)，CAS和synchronized。

  • ArrayList 与 LinkedList

    • 都不能保证线程安全
    • 底层实现：Object数组 vs. 双向链表
    • 插入删除时间复杂度受元素位置的影响 vs. O(1)
    • 支持快速随机访问 vs. 不支持
    • 内存空间占用：列表结尾会预留一定的容量空间 vs. 单个元素消耗更多空间
